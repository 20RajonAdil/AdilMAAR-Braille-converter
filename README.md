# 🧠 AdilMAAR Braille Converter

A modern, clean, and interactive **Text ↔ Braille converter web application** built using HTML, CSS, and JavaScript.

👉 Live Demo:  
https://20rajonadil.github.io/AdilMAAR-Braille-converter/AdilMAAR%20System.html

---

## ✨ Features

- 🔤 Text → Braille conversion
- 🔁 Braille → Text reverse mode
- 🔢 Number support with Braille encoding
- ✍️ Auto-expanding input box
- 📦 Auto-expanding output display
- 🎨 Modern dark UI design
- 📱 Fully responsive (mobile, tablet, desktop)
- ⚡ Instant conversion (no reload)
- 📋 One-click copy result

---

## 🚀 How It Works

1. Type your text in the input box
2. Click **Convert**
3. See the Braille output instantly
4. Use **Reverse Mode** to decode Braille back to text

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (modern responsive design)
- Vanilla JavaScript
